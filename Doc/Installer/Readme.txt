Before running ProtoNow install package, you have to make sure following .NET Framework 4 and update have been installed on your machine:

1. Install Microsoft .NET Framework 4:

Standalone Installer: 
http://www.microsoft.com/en-us/download/details.aspx?id=17718

Or Web Installer
http://www.microsoft.com/en-us/download/details.aspx?id=17851

2. Install Microsoft .NET Framework 4 Update:
http://www.microsoft.com/en-us/download/details.aspx?id=3556
